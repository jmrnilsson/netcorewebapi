# Minimum .NET Core ASP.NET WebApi-example

 * Working example with common commands like;
    * `dotnet test`
    * `dotnet run`
    * `dotnet restore`
 * Xunit
 * Test runner
 * Moq-preview
 * Static files serving
 * HttpClient
 * Async tests
 * WebApi
